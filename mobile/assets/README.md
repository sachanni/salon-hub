# Root Assets Directory

This directory contains app-level assets used by Expo.

## Required Files

Place these files in this directory (not in /src/assets/):

- `icon.png` (1024x1024px) - App icon
- `adaptive-icon.png` (1024x1024px) - Android adaptive icon  
- `splash.png` (1242x2436px) - Launch splash screen
- `notification-icon.png` (96x96px) - Push notification icon

These files are referenced in `app.json` and must exist for the app to build properly.

## Temporary Placeholders

Until you add real assets, you can:
1. Create simple colored placeholders
2. Use online placeholder generators
3. Download from Unsplash/Pexels

See `/mobile/ASSETS_GUIDE.md` for complete specifications and sources.
